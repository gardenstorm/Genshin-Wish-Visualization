# Genshin Wish Visualization
A personal data analysis and visualization project built from five years of gacha pull records. Inspired by Paimon.moe, I wanted to create my own interactive visualizations of my wish history using Python.

The dataset combines a personal Excel spreadsheet (2020–2025) with a Paimon.moe export (2021–2026), merged and cleaned into a unified pipeline.
Current visualizations include:

- Pity frequency histograms
- 5-star and 4-star character pull frequency
- Weapon frequency by rarity and banner
- Stacked bar charts comparing limited and standard banners
- Element-colored treemaps using the Genshin.dev API
- Monthly pull trends by rarity

**Tools**: Python, Pandas, Matplotlib, Plotly, Google Colab, Google Sheets, Microsoft Excel

*AI tools (Gemini, Claude) were used for debugging and refactoring assistance.*

**Status**: Work in progress — interactive web page planned.
